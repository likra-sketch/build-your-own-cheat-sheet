Basic HMTL Document Elements

This table describes basic elements that are needed to properly set up an HTML document.

Tag Name Description
<!DOCTYPE html> Document type declaration This element indicates that the document is an HTML document. It's the initial element in every HTML file.
<html> HTML root Directly follows the document type declaration. Serves as a container for all other HTML elements in the document.
<head> HTML head The head element is a child of the HTML root element. It contains metadata that isn’t directly displayed on the website.
<title> Website title The title element is a child of the HTML head element. It’s responsible for showing the website title in the browser tab display.
<body> HTML body The body element is a child of the HTML root element. It contains all HTML content that can be displayed on a website.

Basic HTML Content Elements

This table describes basic HTML tags that can be used to create a website.

Tag Name Description
<h1>, <h2>, <h3>,
<h4>, <h5>, <h6>		 Heading <h1> to <h6> define HTML headings. Heading importance is indicated by the number in ascending order.
The higher the number, the less important is the heading.
<p> Paragraph The paragraph element contains a block of plain text.
<a> Anchor The anchor element is used to add links to a website. The text of the link is embraced within the opening and closing tag of the element. The link URL is set with the href attribute.
<span> Span The span element is used to separate pieces of text within a paragraph element.
<em> Emphasize The emphasize element is used to display text in italics.
<strong> Strong The strong element is used to display text as bold.
<br> Line break This element adds a line break into the text. Note: the line break element does not have a closing tag.
<ul> Unordered list The unordered list element creates a bullet point list. List elements need to be added by using <li> elements inside the <ul> element.
<ol> Ordered list The ordered list element creates a list of numbered elements. List elements need to be added by using <li> elements inside the <ol> element.
<li> List The list element adds items to a list. It’s used as a child of the <ul> or <ol> elements.
<img> Image The image element adds an image to a website. Note: it does not have a closing tag. Image source/path and alternate text are set with the src and alt attributes.
<video> Video The video element adds a video to a website. Text that will be shown in case the video is not loaded needs to be embraced within the opening and closing tag of the element. Video source/path is set with the src attribute.
<audio> Audio The audio element embeds audio files into a website. Text that will be shown in case the audio file is not loaded needs to be embraced within the opening and closing tag of the element. The element requires a nested <source> element that specifies the path and type of the file via src and type attributes.
<div> Division The division element can be used to group the website into divisions. It serves as a container to group other elements together.
